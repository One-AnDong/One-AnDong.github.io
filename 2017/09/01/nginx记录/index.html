nginx记录

发表于 2017-09-01 | 分类于 前端

环境

环境：macOS

安装方式

brew（brew具体的安装方法请具体google）

安装方法

brew install nginx(brew 是mac非常好用的包管理器，如果你是mac用户建议下载使用)

nginx目录

nginx主目录: /usr/local/Cellar/nginx/1.15.2/
nginx配置文件: /usr/local/etc/nginx/nginx.conf
默认www目录: /usr/local/var/www
/usr/local/Cellar/nginx/1.15.2/html(这个地址是直接映射到www的，www目录是可以自己配置的)

启动和关闭方法

brew:
启动 brew services start nginx
关闭 brew services stop nginx
reload brew services reload nginx

nginx:
启动 ./nginx
关闭 ./nginx -s stop
reload ./nginx -s reload

ps:必须在该目录下使用以上指令 /usr/local/nginx/1.15.2/bin

查看状态

可以使用查看进程的方式查看开启状态
指令: ps aux | grep nginx（ps aux是查看所有进程，grep表示搜索）

mac下软连接使用方式

ln -s 源文件 目标文件
例如你想在当前目录下建立一个文件夹www来连接到nginx的www目录，指令时这样的，在当前目录，敲入 ln -s /usr/local/var/www www 回车，成功后没有任何提示，这样当前目录下就生成了一个www目录，链接到了/usr/local/var/www了
