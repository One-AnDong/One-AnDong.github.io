发表于 2017-03-21 | 分类于 前端

前言

　　今天在技术群里有人在问用css怎么画三角形，所以也就有了这篇小文章，分享用css画三角形的方法，包括实心的和空心的，挺有趣的一个东西，下面我开始介绍这个方法。

正文

　　画三角形的原理，利用盒子模型，令div宽高为0，设置border为实线，设置大小。四个边分别为四个三角形，我们只需要取其中一个，设置颜色，其它border的颜色设置为透明，这样我们就得到一个三角形。参考下图。

　　从左边到右边，div的宽高逐渐减小，我们可以看到，当宽高为0时，出现了4个三角形，代码我就不贴了。下面我写了个小demo可以做参考。
